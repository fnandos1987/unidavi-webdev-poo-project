# unidavi-webdev-poo-project

1) Execução do projeto de Novembro 2017 (projeto feito no NetBeans):
1.1) Descompactar o arquivo Oscar.zip;
1.2) Configurar o Tomcat no projeto;
1.3) Criar uma nova base de dados PotsgreSQL chamada 'oscar';
1.4) Executar o arquivo 'oscar.sql' existente na pasta Oscar\src\conf;
1.5) Executar o projeto.
